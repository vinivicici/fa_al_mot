# Embedding & Clustering Overview

## 프로젝트 목적
본 프로젝트는 패션 상품의 **이미지 및 텍스트(description)**를 기반으로  
사전 정의된 라벨 없이 **상품 간 잠재적 유사 구조를 발견**하는 것을 목표로 한다.  
이를 위해 멀티모달 임베딩과 비지도 클러스터링 기법을 결합하였다.

---

## Embedding 개요
- **모델**: CLIP (image–text joint embedding)
- **입력**: 이미지, 텍스트, 또는 이미지+텍스트 결합
- **의의**: 시각적 유사성과 의미적 유사성을 동일한 벡터 공간에서 비교 가능  
- 임베딩은 L2 정규화를 거쳐 거리 기반 분석에 적합하도록 처리하였다.

---

## Clustering 접근 (총 4가지)

본 프로젝트에서는 임베딩 공간의 구조를 다각도로 검증하기 위해  
서로 다른 특성을 가진 **4가지 클러스터링 기법**을 사용하였다.

1. **K-Means**
   - 거리 기반의 대표적 군집 알고리즘
   - 전체 구조를 빠르게 파악하기 위한 baseline

2. **Gaussian Mixture Model (GMM)**
   - 군집이 하나의 중심이 아닌 확률 분포를 따른다고 가정
   - K-Means 대비 유연한 경계 형성 가능

3. **Agglomerative Clustering**
   - 계층적 병합 방식의 군집화
   - 데이터 간 상대적 거리 구조 분석에 적합

4. **Deep Embedded Clustering (DEC)**
   - 임베딩 학습과 클러스터링을 동시에 최적화
   - 기존 임베딩 기반 방법의 한계를 보완하는 딥러닝 기반 접근

---

## 클러스터 수 결정
클러스터 수 K는 임의로 고정하지 않고 다음 지표를 종합적으로 고려하였다.
- **Elbow Method (Inertia)**
- **Silhouette Score**
- **Gap Statistic**

이를 통해 과도한 분할이나 과소 분할을 피하고,  
임베딩 공간의 구조를 가장 잘 설명하는 K 범위를 탐색하였다.

---

## 전체 파이프라인 요약
1. 데이터 정합성 확보 (CSV–이미지 매칭)
2. CLIP 기반 임베딩 생성
3. (선택) PCA를 통한 차원 축소
4. 4가지 클러스터링 기법 적용
5. 지표 기반 클러스터 수 분석
6. 결과 저장 및 해석
